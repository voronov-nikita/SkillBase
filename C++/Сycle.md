Когда нам известно количество действий, то определенно нужно использовать цикл `for`.

## Цикл `for`


```cpp

#include <iostream>

using namespace std;

int main(){

    for (int i=0; i<100; i++){
        cout << i << endl;
    }

    return 0;
}

```
